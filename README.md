# Data3402-TabularKaggleProject
Here I will be completing the Tabular Kagle Project assigned by Dr. Farbin. I will be cboosing a classifcation problem either provided by Dr. Farbin or one I find interesting for this project. 

## Overview
Predict whether a Santander Bank customer is unsatisfied with their experience
using anonymized transactional and account data. This is a binary classification
problem evaluated on ROC-AUC score.

## Dataset
- 76,020 training samples, 371 anonymized features
- Severe class imbalance: ~96% satisfied (0), ~4% unsatisfied (1)
- No missing values
- Features grouped by prefix: `ind_` (indicators), `num_` (counts), 
  `saldo_` (balances), `imp_` (amounts), `delta_` (changes over time)
